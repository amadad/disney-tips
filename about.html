How It Works

This site automatically collects and organizes Disney World tips from trusted YouTube creators. No more watching hours of videos to find the one tip you need.

The Process

  1. Discover videos — We monitor RSS feeds from top Disney YouTubers to find new content daily.
  2. Fetch transcripts — Video transcripts are pulled automatically using YouTube's caption system.
  3. Extract tips with AI — Google's Gemini AI reads each transcript and pulls out specific, actionable tips.
  4. Categorize & tag — Each tip is tagged by park, category, priority level, and seasonal relevance.
  5. Filter duplicates — Similar tips are grouped so you see unique advice, not repetition.

Tech Stack

TypeScript Vite Gemini AI GitHub Actions GitHub Pages

The pipeline runs daily at 6 AM UTC via GitHub Actions. New videos are processed and tips are updated automatically. The entire site is static HTML — no server, no database, just fast.
